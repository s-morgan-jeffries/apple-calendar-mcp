# Contributing

Thanks for your interest in the Apple Calendar MCP Server!

## Bug Reports and Feature Requests

Please use the [issue templates](https://github.com/s-morgan-jeffries/apple-calendar-mcp/issues/new/choose) to file bug reports or request features. Include as much detail as you can — reproduction steps, macOS version, and error messages are especially helpful.

## Code Contributions

This is a personal project and is not accepting pull requests. You're welcome to fork and adapt for your own use.
